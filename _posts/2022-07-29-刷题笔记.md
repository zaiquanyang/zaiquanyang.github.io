## 🍉 动态规划

####  [300.最长递增子序列](https://leetcode.cn/problems/longest-increasing-subsequence/)

解法一：完全基于动态规划
```c++
class Solution {
public:
    int lengthOfLIS(vector<int>& nums) {
        int len = nums.size();
        // 定义dp数组并初始化
        vector<int> dp(len, 1);
        // 更新dp数组
        for(int i=1; i<len; i++) {
            for(int j=0; j<i; j++) {
                if(nums[i] > nums[j]) {
                    dp[i] = max(dp[i], dp[j]+1);
                }
            }
        }
        int res = 0;
        for(int k=0; k <len; k++){
            if(dp[k] > res) res = dp[k];
        }
        return res;
    }
};
```
